女性必見！ジュエリー感覚でつけられる時計特集【4選】

2016.6.23 スポーツ remove_red_eye3606

腕時計だけどブレスレット感覚でつけられる時計や、ネックレス、ピアスなどジュエリーとのコーディネートが決まる時計をご紹介します。

ジュエリーとしての時計の存在価値

via encrypted-tbn2.gstatic.com
時計といえば時間を見るもの、仕事をする上でのマストアイテム、男性だとさらにこだわりが増し、腕時計をつけることがステータスにもなっています。

女性だと時計の機械などに興味がある方は少ない印象で、様々なブランドから出ている時計もほとんどクォーツ。

やはり選ぶ基準は見た目のかわいさや、着用時に手元がきれいに見えるか、などジュエリーを購入するときと同じような感覚で選ぶことが多くなるかと思います。
今回は女性らしさ抜群の時計でジュエリーとも相性のいい時計をご紹介していきます。

1.カルティエ ミニパンテール YG

cartier ミニパンテール イエローゴールド

via encrypted-tbn0.gstatic.com
華奢な時計といえば、こちらのミニパンテール。
ケースの小ささだけでなく、リューズが裏蓋にあることで出っ張りのないスタイリッシュなデザイン。
ケースの厚みもかなり薄いので、着用するとブレスレットのようです。

デザインも様々なものがあり、ベゼルにダイヤが一重、二重のタイプや、ブレスまで全てにダイヤが施されているもの、アールデコ調のデザインものなど、挙げるとキリがないくらいです。

某モデルさんなどもつけていたことで有名で現在では廃盤モデル。
中古での出回りは比較的多いので自分に合うお気に入りの1本も探しやすいと思います♪

2.ブルガリ セルペンティ

BVLGARI セルペンティ SS/PG ST35BSPG.1T

via g.kame-kichi.com
ブルガリ特有の手に巻きつけるタイプのセルペンティ。
蛇をモチーフにしたデザインで手につけるとセクシーな印象に...
素材やデザインも様々なタイプが出ており、腕周りが二重になっているものもあり腕につけた存在感は抜群！
コンビモデルだと手持ちのプラチナやPG、YG、WG素材のどんなものでも相性バッチリ♫
同じシリーズのリングやピアスも展開しているので全身コーデも可能♡

腕が華奢に見えます🎶

ついつい手元に目がいってしまいます♪

3.フレッド フォース10 ウォッチ

FRED フォース10 FD070111A002 ステンレス/ダイヤ

via www.bettyroad.co.jp
ブレスレットで有名なフレッド フォース10。
実は同じデザインで時計も展開されています。一つで時計とブレスの1本2役で、つけているだけでオシャレ度が増します。

ケーブル部分も様々な色が展開しており、気分によって付け替えも簡単に出来ちゃいます！
男性でも時計とフレッドのブレスの重ね付けをしてる方も多く、おしゃれに敏感な男性にも一目置かれる存在に！？

ショパール ハッピースポーツ

CHOPARD ハッピースポーツ 27/8250-3022

via tshop.r10s.jp
言わずと知れた高級ブランドショパール。
ムービングダイヤのデザインでおなじみのブランド。
こちらの時計もムービングダイヤが５石入っており時間を見るたびについつい手を振って動きを楽しんでしまいます。

これまでに挙げてきたものと違うのが、一番腕時計らしいフォルムをしているのでシーンを問わずにつけていただける時計です。

また同じムービングダイヤを採用したジュエリーの展開も多いのでオンタイムは時計のみ、オフタイムはジュエリーと合わせて、といったコーデも可能。
ムービングダイヤは3石〜7石、魚型や月や星の形わしたものもあり、遊び心もいっぱいな1本です。
